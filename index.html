簡介

本書涵蓋「自然語言、程式語言、標記語言」處理等領域，包含語法、語意、編譯、計算語言學、自然語言理解、交談系統、機器翻譯等主題。

本書為創作共用作品，採用 創作共用：姓名標示、非商業性、相同方式分享授權 ，包含很多簡易的實作程式，讓您能瞭解如何實作自然語言處理系統，而不僅僅是理論陳述而已，這些程式碼都可以直接在 github 上下載。
